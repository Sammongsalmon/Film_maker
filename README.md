# 필름 프레임 메이커

브라우저 안에서 사진을 합성하고 PNG/JPG로 저장하는 정적 웹앱입니다. 서버나 빌드 과정 없이 GitHub Pages에 바로 배포할 수 있습니다.

## 포함된 메이커

- 영화 장면: 대표 화면비, 레터박스 3가지 설정 방식, 자막 템플릿/직접 편집
- 카메라 뷰파인더: 가로·세로 벡터 프레임, 디지털/필름 모드, REC·배터리·삼분할 가이드, 표시색/프레임색 분리
- 35mm 소형 필름롤: 1~36컷, 줄바꿈, 줄 어긋남, 필름 베이스 색상, 그림자, 배경/그라데이션/패턴, 줄 단위 자유변환·레이어 순서
- 120 중형 필름: 가로·세로 벡터 프레임, 얇은 프레임과 회전되는 필름 표기

## 공통 기능

- 사진 드래그 이동, 휠 확대/축소, 슬라이더 이동/회전, 90도 회전
- 색감 필터, 필름 빛샘, 절차형·사용자 이미지 오버레이, 그레인
- PNG 투명/흰 배경, JPG 저장, 파일명과 출력 배율 설정
- 조선일보명조, Pretendard Variable 포함 및 사용자 폰트 업로드
- 새로고침 후에도 사진·효과·배치·내보내기 설정을 브라우저 IndexedDB에 자동 저장/복원
- 실행 취소·다시 실행 버튼과 `Ctrl/⌘+Z`, `Ctrl/⌘+Shift+Z`, `Ctrl+Y` 단축키
- 실행 취소·다시 실행과 내보내기 설정을 모드 탭 위의 고정 도구 영역에 배치
- 소형 필름 효과 적용 범위와 컷별 랜덤 빛샘 옵션을 효과 스택에서 설정
- 모든 처리는 로컬 브라우저에서 수행

## GitHub Pages 배포

1. 이 폴더의 파일을 저장소 루트에 업로드합니다.
2. GitHub 저장소의 **Settings → Pages**로 이동합니다.
3. **Deploy from a branch**를 선택하고 `main` 브랜치의 `/ (root)`를 지정합니다.
4. 저장 후 표시되는 Pages 주소로 접속합니다.

별도의 `npm install` 또는 빌드 명령은 필요하지 않습니다.

## 파일 구조

```text
index.html
css/styles.css
js/app.js
assets/fonts/
assets/references/
.nojekyll
```

`assets/references/`의 이미지는 벡터 프레임 제작 참고용이며 웹 화면에는 직접 합성되지 않습니다.

## 작업 저장 방식

현재 작업은 같은 브라우저와 같은 배포 주소의 IndexedDB에 자동 저장됩니다. 사진과 사용자 오버레이 이미지, 사용자 폰트도 함께 복원됩니다. 브라우저의 해당 사이트 데이터가 삭제되거나 시크릿 창을 닫으면 저장 내용이 사라질 수 있습니다. 실행 취소 기록은 현재 탭에서 최대 40단계까지 유지되며, 새로고침 후에는 저장된 최종 작업부터 새 기록이 시작됩니다.

## 소형 필름 줄 자유변환

`스트립 배열 → 자유변환 모드`를 켜면 줄바꿈 기준으로 연결된 한 줄이 하나의 객체가 됩니다. 미리보기에서 줄을 선택해 이동하고, 모서리 핸들로 확대·축소하고, 위쪽 원형 핸들로 회전할 수 있습니다. 설정 패널에서는 X/Y 이동, 크기, 회전값을 직접 입력하고 줄을 한 단계 앞으로 또는 뒤로 보낼 수 있습니다.

## v8 자막 폰트 수정

- 영화 자막에 볼드 옵션을 제공합니다.
- 프리텐다드 굵기 조절은 프리텐다드 선택 시에만 표시됩니다.
- 프리텐다드 선택 후 굴림·돋움·명조·조선일보명조·사용자 폰트로 다시 변경되지 않던 폰트 전환 문제를 수정했습니다.

## v9 소형 필름 베이스 색상

소형 필름 탭의 `대지와 필름 크기` 블록에서 필름 베이스 색상을 변경할 수 있습니다. 프레임 면과 사진을 넣지 않은 컷의 출력 색상이 함께 바뀌며, 기존 저장 작업에는 기본값 `#2b2927`이 자동 적용됩니다.

## v10 수치 입력과 컬러 선택 개선

- 확대/축소, 컷 수, 줄바꿈 기준 컷을 포함한 모든 숫자 입력칸은 기존 값을 전부 지운 뒤 새 숫자를 입력할 수 있습니다.
- 숫자를 편집하는 동안 빈 값이나 `-` 같은 중간 입력을 강제로 최솟값으로 되돌리지 않습니다.
- `Enter` 또는 포커스 이동 시 허용 범위에 맞춰 최종값을 확정하며, `Esc`를 누르면 편집 전 적용값으로 돌아갑니다.
- 컬러 스와치를 누르면 해당 입력 바로 주변에서 브라우저의 HSV 컬러피커가 열리며, 패널에 팔레트가 항상 노출되지 않습니다.
- HEX 입력칸에 `#RRGGBB` 형식으로 직접 입력하는 방식도 함께 지원합니다.

## v11 변경 사항
- 제공된 텍스처 ZIP의 19개 이미지를 기본 텍스처로 내장하고 snow, rain, bokeh, scratch, grain, particle 순으로 다시 라벨링했습니다.
- 소형 필름 사진 영역을 3:2로 확대하고 좌우 프레임 여백을 기존의 약 40%로 줄였습니다.
- 소형 필름 삼각형을 컷당 2개로 정리하고 여러 컷에서도 일정 간격으로 이어지도록 배치했습니다.
- 연결된 필름 한 줄을 하나의 합성 객체로 렌더링하여 컷 사이 그림자와 빈 픽셀 경계가 생기지 않도록 수정했습니다.


## v12 변경 사항

- 효과 카드마다 `사진 영역만` 또는 `전체 영역` 적용 범위를 선택할 수 있습니다.
- 효과 카드 머리글의 위·아래 화살표로 그레인, 텍스처, 빛샘, 색감 필터의 합성 순서를 자유롭게 바꿀 수 있습니다.
- 제공된 텍스처 19개를 `snow`, `rain`, `bokeh`, `scratch`, `grain`, `particle` 묶음으로 정리했습니다. 텍스처는 선택 범위를 빈틈없이 덮는 최소 확대율이 자동 적용되며 위치와 확대율을 조절할 수 있습니다.
- 뷰파인더 모드는 디지털/필름 버튼으로 즉시 전환됩니다. 텍스트와 프레임·아이콘 색상을 분리했고 필름 옐로 색상을 추가했습니다. 디지털 배터리는 4K 표시와 우측 정렬되며 프레임·아이콘 색상을 따릅니다.
- 중형 필름 프레임을 얇게 조정하고 삼각형을 숫자 옆에 나란히 배치했습니다.
- 소형 필름은 컷마다 삼각형 두 개를 일정 간격으로 배치하고, 3:2 사진 영역을 넓혔으며, 연결된 한 줄 전체에 한 번만 그림자를 적용해 컷 경계의 틈과 그림자선을 제거했습니다.

## v13 변경 사항

- 소형 필름 자유변환 모드의 모서리와 회전 핸들을 미리보기 배율에 맞춰 충분히 크게 표시하고, 실제 클릭·터치 판정 범위도 넓혀 캔버스에서 직접 확대·축소·회전할 수 있도록 수정했습니다.
- 중형 필름의 숫자와 삼각형을 사진 영역이 아닌 프레임 띠의 중앙선 위로 이동했습니다. 가로·세로 전환 시에도 프레임 전체 회전 규칙을 그대로 따릅니다.
